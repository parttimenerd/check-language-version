### Summary

Adds `TextInputStream` (alpha3 era) for reading textual data from a stream.

### Details

HotJava/Java 1.0-alpha3 listed `java.io.TextInputStream` as a new class.

Modern Java typically reads text via `InputStreamReader` and `BufferedReader`.
